# 📋 Panduan Penggunaan Sistem Hutatus Coffee

Halo! Panduan ini dibuat untuk membantu Anda mengelola kafe digital Hutatus Coffee dengan mudah. Tidak perlu keahlian khusus, cukup ikuti langkah-langkah di bawah ini.

---

## 1. Cara Masuk ke Halaman Admin (Halaman Khusus Pemilik)

Untuk melihat pesanan atau mengatur menu, Anda harus masuk ke "Admin Area":
1. Buka website Hutatus Coffee.
2. Geser (scroll) ke bagian paling bawah website (**Footer**).
3. Klik tulisan **"Admin Area"** yang ada di pojok bawah.
4. Masukkan PIN Rahasia Anda (Default: **252525**).
5. Klik **"Masuk Dashboard"**.

---

## 2. Cara Mengatur Menu & Harga

Di menu **"Produk"**, Anda bisa melakukan hal-hal berikut:

### Menambah atau Mengubah Menu
Setiap menu bisa Anda atur kategorinya (**Kopi, Non-Kopi, Camilan, atau Dessert**).

### Mengaktifkan Pilihan (Variasi)
Jika Anda menjual kopi, Anda bisa menyalakan tombol:
- **Pilihan Size**: Jika dinyalakan, Anda bisa mengisi 3 harga berbeda untuk ukuran **Kecil, Sedang, dan Besar**.
- **Pilihan Gula & Es**: Jika dinyalakan, pelanggan bisa memilih (Tanpa Gula, Sedikit Es, dll) saat memesan.

---

## 3. Cara Mengelola Pesanan Masuk

Setiap ada pesanan baru, Anda bisa melihatnya di menu **"Pesanan"**:
1. **Status Menunggu**: Pesanan baru masuk, tapi pelanggan belum konfirmasi pembayaran.
2. **Status Diproses**: Klik tombol ini jika Anda sudah menerima bukti transfer di WhatsApp dan mulai menyeduh kopi.
3. **Status Selesai**: Klik ini jika kopi sudah diserahkan ke pelanggan.

---

## 4. Cara Mengganti Gambar QRIS Pembayaran

Jika Anda punya kode QRIS baru, Anda tidak perlu memanggil teknisi. Anda bisa menggantinya sendiri:
1. Siapkan foto QRIS Anda di komputer/HP.
2. Beri nama file foto tersebut: **`qris.png`**.
3. Masukkan (copy-paste) file tersebut ke dalam folder bernama **`public`** di dalam folder project ini.
4. Gambar QRIS di website akan otomatis berubah mengikuti foto terbaru Anda.
